7-3 Transform element using rotate scale translate

This property allows you to rotate, scale, move, skew, etc., elements.

  • scale = zoom in out kora (0.75)
  • Rotate = rotate(45deg)
  • translate = reposition an element horizontal or vertical
  • All action can take x-axis and y-axis
  • Example: translate(50px, 50px)
